Колонки CSV-файла

Файл должен содержать заголовки в первой строке. Используемые колонки:

Новый формат

Заголовок в файле Поле в базе Примечание
Название Название ★ Основной идентификатор записи
Аббревиатура — Игнорируется при импорте
Регион Регион + Город Город извлекается автоматически («г. Москва» → «Москва»)
Адрес Адрес Если город не найден в регионе — ищется в адресе
Сайт Сайт https:// добавляется автоматически если отсутствует
Email Email
Телефон Телефон
Тип, Федеральный округ, Руководитель, ОГРН, ИНН, Статус лицензии — Игнорируются при импорте
Старый формат (Рособрнадзор) — также поддерживается
Название (краткое) Название ★ Ключевое поле. Если пусто — берётся полное
Название (полное) Полное название Официальное наименование

★ — обязательное поле

Логика при повторном импорте

  • Если университет с таким названием уже есть в базе — его данные обновляются.
  • Если не найден — создаётся новая запись.
  • Поля has_dormitory, has_military, photo, description при импорте не затрагиваются.
