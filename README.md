# EduSupport — Информационная система для абитуриентов

Веб-приложение на Django, которое помогает абитуриентам выбирать университеты и образовательные программы на основе результатов ЕГЭ.

---

## Возможности

- 🔍 **Каталог университетов** — поиск и фильтрация по городу, наличию общежития и военной кафедры, наукометрическим показателям (h-index, цитирования)
- 🎓 **Каталог программ** — фильтрация по уровню подготовки, форме обучения, университету, предметам ЕГЭ
- 🧮 **Калькулятор шансов** — подбор программ по баллам ЕГЭ с ориентировочным показателем поступления
- ⚖️ **Сравнение программ** — сопоставление до 3 программ по единому набору параметров
- ❤️ **Избранное** — сохранение программ для быстрого доступа
- 📋 **Справочник абитуриента** — этапы поступления, документы, категории льготников, FAQ

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/SkywalkerERR/edusupport-web-app.git
cd edusupport-web-app
```

### 2. Создать и активировать виртуальное окружение

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

Создать файл `.env` в папке `abiturient_project/`:

```env
SECRET_KEY=your_secret_key
DEBUG=True
DB_NAME=edusupport
DB_USER=postgres
DB_PASSWORD=your_password
DB_HOST=localhost
DB_PORT=5432
```

### 5. Применить миграции

```bash
cd abiturient_project
python manage.py migrate
```

### 6. Создать суперпользователя

```bash
python manage.py createsuperuser
```

### 7. Запустить сервер

```bash
python manage.py runserver
```

Приложение доступно по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000)  
Панель администратора: [http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin)

## Импорт данных

### Университеты из CSV

```bash
python manage.py import_universities vuz_contacts_test.csv
# С флагами:
python manage.py import_universities vuz_contacts_test.csv --update    # обновить существующие
python manage.py import_universities vuz_contacts_test.csv --dry-run   # проверка без записи
python manage.py import_universities vuz_contacts_test.csv --encoding cp1251
```

Также доступен импорт через **Admin → Университеты → Импорт CSV**.

### Наукометрические данные (OpenAlex)

```bash
python manage.py import_openalex_csv universities_with_counts.csv
python manage.py import_openalex_csv universities_with_counts.csv --dry-run
python manage.py import_openalex_csv universities_with_counts.csv --threshold 0.4
```

## Ключевые моменты использования

### Калькулятор шансов

1. Зарегистрируйтесь и войдите в личный кабинет
2. Перейдите в **Профиль → Баллы ЕГЭ** и введите свои результаты по каждому предмету
3. Откройте **Подбор по баллам** — система автоматически подберёт подходящие программы
4. Используйте фильтры: минимальный балл, наличие бюджетных мест, общежитие, военная кафедра
5. Сортируйте результаты по показателю поступления, стоимости или количеству мест

> ⚠️ Показатель поступления является **ориентировочным** и не представляет собой статистическую вероятность.

### Сравнение программ

- Добавить программу в сравнение можно с любой карточки (кнопка **Сравнить**)
- Максимум **3 программы** одновременно
- Список сравнения сохраняется при навигации по сайту
- При сравнении программ разных уровней подготовки система выводит предупреждение

### Структура программы

Каждая образовательная программа содержит:
- **Обязательные предметы ЕГЭ** — необходимы для поступления
- **Предметы ЕГЭ по выбору** — один из группы на выбор абитуриента
- **Вступительные испытания** — внутренние экзамены университета (при наличии)
- **План приёма** — бюджетные/платные места, стоимость, проходные баллы по годам

### Минимальная сумма баллов (порог)

Рассчитывается автоматически по формуле:

```
сумма(мин. баллов обязательных предметов) + мин(мин. баллов необязательных предметов)
```
